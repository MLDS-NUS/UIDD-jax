# Examples
This folder contains example scripts and resources.


## reduced_polymer_dynamics_F_HD_scale
Learning reduced variable of polymer dynamics under **varying [force] and fixed [temperature=1]** conditions, employing [Helmholtz–Hodge Decomposition] with a scaled potential function.

## reduced_polymer_dynamics_F_T1
Learning reduced variable of polymer dynamics under **fixed [force=1.62] and varying [temperature]** conditions, employing [Helmholtz–Hodge Decomposition] with a scaled potential function.

## reduced_polymer_dynamics_F_T23
Learning reduced variable of polymer dynamics under **fixed [force=23.10] and varying [temperature]** conditions, employing [Helmholtz–Hodge Decomposition] with a scaled potential function.
